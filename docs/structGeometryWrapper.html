Detailed Description

template<typename GeometryType>
struct GeometryWrapper< GeometryType >

Wrapper class for the Geometry objects.

Contains only the definitions of the functions so that the derived geometry classes can inherit and overload them.

Warning
Never should anyone define an object of this type and try to utilize it (undefined behaviour)
