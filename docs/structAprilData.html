Detailed Description

Holds all the APRIL related metadata for a single object.

This struct contains two lists and their sizes, for APRIL's A-list and F-list. The intervals' start,end are stored as consecutive numbers. Starting from the first position, every two consecutive numbers represent an interval's [start,end).
